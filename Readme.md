## 📌 Project Overview

This project is an interactive **Power BI dashboard** developed to analyze **Artificial Intelligence & Machine Learning job market trends**.

The dashboard provides insights into:

- 📈 Salary trends
- 🌍 Job locations
- 🏢 Company hiring patterns
- 💼 Experience level distribution
- 🧠 Skill demand
- 🏠 Remote work opportunities

The objective of this project is to demonstrate data visualization, dashboard designing, KPI creation, and business insight generation using **Power BI**.
